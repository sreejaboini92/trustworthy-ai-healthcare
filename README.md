# Trustworthy AI in Healthcare

## Overview
A Flask-based healthcare AI dashboard demonstrating key principles of Trustworthy AI.

## Features
- Bayesian Neural Network inspired uncertainty estimation
- Explainable AI (SHAP-style feature importance)
- Human-in-the-loop decision making
- Federated Learning simulation (FedAvg)
- Bias and Fairness Monitoring Dashboard

## Technologies Used
- Python
- Flask
- NumPy
- SHAP
- HTML
- Tailwind CSS
- JavaScript

## Project Workflow
Patient Data
→ Risk Prediction
→ Confidence Estimation
→ Human Review Check
→ Explainable AI Output
→ Clinical Decision Support

- Real healthcare datasets
